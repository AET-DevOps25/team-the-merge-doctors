## Installation

1. Create a virtual environment 
```
python3 -m venv .venv
```

2. Activate virtual environment
```
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

2. Install dependencies
```
pip install -r requirements.txt
```
